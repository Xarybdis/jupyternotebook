For this task, Somerville Happiness Survey dataset is chosen. This dataset contains 143 instances, 7 integer attributes(6 conditional, 1 decision) with values 0 or 1. Attribute Info:

  • D = decision attribute (D) with values 0 (unhappy) and 1 (happy)
  • X1 = the availability of information about the city services
  • X2 = the cost of housing
  • X3 = the overall quality of public schools
  • X4 = your trust in the local police
  • X5 = the maintenance of streets and sidewalks
  • X6 = the availability of social community events
Attributes X1 to X6 have values 1 to 5.

#replacing with mode with respect to decision class
import random
def repl(data):
    for i in np.arange(data.shape[0]):
        for j in np.arange(data.shape[1]-1):
            if pd.isnull(data.iloc[i,j])== True:
                b=data.iloc[i,-1]
                c=[]
                for k in np.arange(data.shape[0]):
                    if data.iloc[k,-1]==b:  
                        if pd.isnull(data.iloc[k,j])==False:
                            c.append(data.iloc[k,j]) 
                if len(c)!=0: 
                    values, counts = np.unique(c, return_counts=True)
                    max_count_index = np.argmax(counts) 
                    c=values[max_count_index]
                    data.iloc[i,j]=c
                else:
                    data.iloc[i,j]=random.choice(data.iloc[:,j].mode())
    return data

# normalising data numerical data sets using min-max
def norm(data):
    for column in data.columns:
        if isinstance(data[column][0],str)==False and column != 'D':
            data[column]=(data[column]-data[column].min())/(data[column].max()-data[column].min())
    return data

Correlation

Attributes might be dependent on each other, in that case it would be redundant to have both variables in the model. For preparation step involves removing those variables, so that will reduce the effect of other variables on the model to determine how variables are dependent on each other correlation is calculated.

def co(data):
    for column in data.columns:
        if isinstance(data[column][0],str)==True:
            data.drop([column],axis=1)
    cor=data.corr()
    return cor
